No cookies

We do not use cookie banners because we don’t set tracking cookies.

How calculators work

Calculations run locally in your browser. Your inputs are not sent to a server by these tools. Some pages may optionally encode inputs into the page URL (so you can share a scenario or jump to an explainer) — this is visible in your address bar.

Server logs

Like most websites, the hosting provider may log basic request data (e.g., IP address, user agent, requested URL) for security and performance. We don’t attempt to link this to your identity.

Contact

If you want something removed or have questions, contact the site owner via the GitHub repository issues.
